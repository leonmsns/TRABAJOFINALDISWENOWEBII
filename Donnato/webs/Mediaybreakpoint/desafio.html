Breakpoints y Media Queries

Hecho por: Donnato Mesones

Sus significados

Los breakpoints y las media queries son conceptos fundamentales en el diseño web responsivo.
Breakpoints: Son los puntos de ruptura en los que la pantalla cambia de layout en el diseño responsive. Son las medidas de anchura en donde se realizan saltos para el diseño responsive y se aplican los estilos CSS concretos para unas determinadas media queries. En otras palabras, los breakpoints son los puntos en los que la pantalla cambia de diseño.
Media Queries: Las Media queries permiten aplicar estilos CSS según el tipo general de un dispositivo (como impresión o pantalla) u otras características como la resolución de la pantalla o el ancho del viewport del navegador. Se utilizan para aplicar estilos condicionalmente utilizando las reglas de arroba CSS @media e @import. También se utilizan para segmentar medios específicos para style, link, source y otros HTML con el atributo media.
